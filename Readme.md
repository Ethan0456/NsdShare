# NSDSHARE

* NsdShare is a Socket based File Sharing App designed in Kotlin with Jetpack Compose.

Download Link : [Download](https://drive.google.com/file/d/1CYdfgul_A0rZnSEBigLOtSEWNlGUEDkj/view?usp=share_link)
